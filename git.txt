1 查看远程分支 
$ git branch -a   

2 查看本地分支 
$ git branch   

3 创建分支 
$ git branch test   

线面是把分支推到远程分支  
$ git push origin test  

4 切换分支到test 
$ git checkout test   
   

M 表示cong 原来分支（上一次修改没有提交br-2.1.2.2）带过来的修改
5 删除本地分支  
git branch -d xxxxx 
 

6 查看本地和远程分支  -a。前面带*号的代表你当前工作目录所处的分支 
$ git remote  -v   

7 删除远程版本
git push origin :br-1.0.0  

8 删除远程分支  
git push origin :branch-name
